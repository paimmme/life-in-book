# 参与贡献

Life in Book 是一个开放的文学人物心理侧写项目。不需要会写代码也能参与。

## 可以做什么

- **提名作品。** 在 Issues 中提交你想看到人物心理分析的作品。
- **补充原文引用。** 读到某个场景觉得现有分析没有用到关键台词？提交 PR 补充。
- **修正分析。** 觉得某个心理推断有问题？指出问题并提供你的观点。
- **提交新档案。** 用 AI 辅助 + 人工审校的方式，为一本书生成完整的三层分析。
- **新增理论框架。** 你是心理学专业背景？建议新的分析维度。
- **代码贡献。** 改进网站功能、新增可视化组件。
- **翻译。** 将现有中文档案改写为英文版（或反过来）。

## 档案撰写指南

每份人物档案是一个 Markdown 文件，包含 YAML frontmatter（定量数据）和正文（定性分析）。

### 基本规则

1. 每一条心理推断必须有原文引用支撑
2. 每条声明必须标注使用的心理学框架
3. 区分"文本支持的结论"和"合理推测"
4. 定量评分需要锚定在全体已分析人物的分布上

### 模板

见 `prompts/` 目录下的 AI 生成提示词模板。

## 提交方式

```
Fork 仓库 → 创建分支 → 按模板编写/修改 Markdown
    ↓
提交 PR
    ↓
审核：
  - 心理推断是否有原文支撑？
  - 理论框架使用是否恰当？
  - 置信度标签是否合适？
    ↓
合并 → 自动部署
```

## 内容许可

所有人物档案、心理分析、时代分析和作者分析均按 CC BY-NC-SA 4.0 发布。
