# Life in Book

**读懂一个人物，理解一本书。**

Life in Book 是一个文学人物心理侧写项目。用心理学框架深度剖析虚构人物，用结构化档案回答那个在阅读中挥之不去的问题——"这个人为什么会这样？"

## 三层分析

| 层级 | 分析对象 | 核心问题 |
|------|---------|---------|
| L1 · 人物侧写 | 虚构人物 | 心理构造是怎样的？行为背后的动机是什么？ |
| L2 · 时代语境 | 故事所属的社会/历史背景 | 时代给人物施加了什么约束？ |
| L3 · 作者表达 | 真实作者 | 作者的个人经历如何映射进作品？ |
